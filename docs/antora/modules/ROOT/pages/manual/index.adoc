This page is the shortest operator path through `cx`.
Use it when you already know you are running the tool and need commands, assurance levels, and workflow choices instead of the full conceptual model.

== Primary operator surfaces

[cols="1,3"]
|===
| Surface | Use it for

| Live workspace
| `cx mcp`, `cx doctor mcp`, `cx audit summary`

| Durable cognition
| `cx notes`, `cx notes check`, repository memory hygiene

| Native proof path
| `cx inspect`, `cx bundle`, `cx validate`, `cx verify`, `cx extract`

| Adapter/oracle diagnostics
| `cx adapter *` for expert parity checks and reference-oracle inspection only
|===
