= Rust Template
:page-layout: default


++++
<p>The Rust <code>cx init</code> template is designed for Cargo workspaces with a richer
local verification gate than the basic templates.</p>
<h2>Generated Targets</h2>
<ul>
<li><code>make build</code> → <code>cargo build --release</code></li>
<li><code>make test</code> → <code>cargo test</code></li>
<li><code>make check</code> → <code>cargo check</code></li>
<li><code>make verify</code> → <code>cargo fmt --check</code>, <code>cargo clippy --all-targets -- -D warnings</code>, <code>cargo test</code>, <code>cargo build --release</code></li>
<li><code>make certify</code> → <code>verify</code></li>
</ul>
<h2>MCP Overlay</h2>
<p>The generated <code>cx-mcp.toml</code> exposes:</p>
<ul>
<li><code>src/**</code></li>
<li><code>tests/**</code></li>
<li><code>benches/**</code></li>
<li><code>Cargo.toml</code></li>
<li><code>Cargo.lock</code></li>
<li><code>README.md</code></li>
</ul>
<p>It excludes <code>target/**</code> so build artifacts do not leak into the default
authoring view.</p>
