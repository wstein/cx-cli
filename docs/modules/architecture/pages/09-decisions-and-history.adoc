== Source of historical pressure

Use promoted notes and release docs for durable decision history.
Keep exploratory or transitional reasoning in `notes/`.

== Promotion Rule

A note becomes architecture documentation only when it states a durable
decision, invariant, mechanism, or failure mode that operators should
depend on. Exploratory reasoning can remain valuable without being
promoted into the curated docs surface.
