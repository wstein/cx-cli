The repository is no longer just producing static bundles; it is participating in agentic workflows. `cx mcp` starts the CX MCP server with `cx-mcp.toml` when present, `cx doctor mcp` proves the inherited MCP scope, and `cx doctor secrets` scans the master list before an agent trusts the workspace.
