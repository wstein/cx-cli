0.4.0 makes Vitest coverage the authoritative coverage-reporting lane for release assurance, while Bun remains part of the execution and compatibility matrix. This distinction matters because operators need one reporting surface for HTML, LCOV, JSON summary, Pages publishing, and release gating, even though the repository still verifies behavior across the Bun-first runtime path.
