# Homebrew Tap

`wstein/homebrew-tap` hosts the Homebrew formula used to install `@wsmy/cx-cli`.
This tap is updated from tagged releases in this repository by the combined release workflow. The formula is generated from a locally packed release tarball and installed with Homebrew's Node integration.
