# npm Package Page

`https://www.npmjs.com/package/@wsmy/cx-cli` is the public npm package listing for the CLI.
The release workflow publishes the packed tarball to this package before generating the Homebrew formula.

The package page represents the shipped runtime artifact, not the optional
reference oracle tooling:

- native proof-path runtime is included
- no runtime Repomix fork dependency is shipped
- adapter/oracle behavior remains test and diagnostic surface only
