0.4.0 is the first release where `cx` should be explained directly as a three-surface system: live MCP hypothesis generation, durable repository cognition, and proof-grade artifact bundling. The release contract needs to say this explicitly so operators do not infer stability or trust semantics only from implementation maturity or scattered documentation.
