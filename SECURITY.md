# Security Policy

## Supported Versions

The active development branch and the latest tagged release are supported for
security fixes.

## Reporting a Vulnerability

If you find a security issue in `cx-cli`, open a private security advisory on
GitHub or contact the maintainers directly before publishing details.

Please include:

- a short description of the issue
- the affected command or file path
- reproduction steps, if available
- whether the problem affects the CLI, docs, or workflow automation

We will triage reports promptly and coordinate disclosure timing before any
public write-up.
